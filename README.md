# NotionCloud - ScoreCenter Elite 1 Féminine

Mini projet : API Node/Express (MySQL) + front statique pour le championnat Elite 1 féminin.


## Endpoints
- GET /api/healthz  → { "ok": true }
- GET /api/matches  → liste JSON des matchs

## Front
- `front/index.html` → accueil (prochain match / dernier résultat)
- `front/resultats.html` → tableau des matchs
